## Instalation

To install the dotfiles and get them on your system run the following commands.

```shell
pacman -Sy git
git clone --depth 1 https://github.com/martprenger/hyprdots
cd ~/hyprdots/Scripts
./install.sh
```

## Updating dotfiles

If you just want to update the dotfiles after you update some config files run the following commands

```shell
cd ~/hyprdots/Scripts
./install.sh -c
```
